### 服务列表
1. 用户中心服务
2. APP 管理服务
3. API 接口服务
4. ...

### 主流框架 & 特性

- Spring Boot 2.7.12
- Spring MVC
- Spring 事务注解
- Spring Cloud
- Spring Cloud Alibaba
- Spring AOP
- Spring Scheduler 定时任务
- MyBatis + MyBatis Plus
- Nacos v2.2.0
- Redis v7.2.5
- Sa-token

## 快速上手

1. 克隆此仓库

```bash
git clone https://github.com/yovvis/easy-open
```

2. 安装依赖（docker 启动开发环境）
